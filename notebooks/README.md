# Ноутбуки с примерами

В папке `notebooks` собраны актуальные Jupyter-ноутбуки с примерами использования библиотеки `manuscript-ocr`.

Начиная с версии `0.1.11` API проекта было изменено, поэтому примеры в корне папки `notebooks` относятся к актуальной версии библиотеки.

Если требуется использовать версию `0.1.10`, следует обращаться к ноутбукам в папке `notebooks/v0_1_10`, где сохранены варианты примеров, согласованные с API версии `0.1.10`.
